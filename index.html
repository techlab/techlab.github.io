jQuery Smart Wizard

The awesome jQuery step wizard plugin with Bootstrap support.

Smart Wizard is a flexible and heavily customizable jQuery step wizard plugin with Bootstrap support. It is easy to implement and gives a neat and stylish interface for your forms, checkout screen, registration steps etc.

  • Bootstrap 3 & 4 support
  • Responsive themes
  • Heavily customizable toolbar
  • Theme support with various themes included
  • Enhanced event support
  • Ajax content loading
  • Keyboard navigation
  • Compatible with latest jQuery versions

jQuery Smart Cart

jQuery Shopping Cart plugin with PayPal payment support

Create interactive and user friendly shopping cart with minimal code. Built-in support for form submit, Ajax submit and PayPal submit.

  • In-built PayPal, Ajax and form submit methods
  • Bootstrap support
  • Compatible with latest jQuery versions
  • Public methods for external function call
  • Enhanced event support
  • Clean and compact design

jQuery Smart Tab

Flexible jQuery Tab Control plug-in

Flexible jQuery plugin for tabbed interface. Load ajax contents, have a verity of themes, cool animation effects, event supports, easy to customize.
